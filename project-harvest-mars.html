Harvest Mars

October 2023

Harvest Mars is a game created during the Ludum Dare 52 game jam, with the theme "Harvest," in 48 hours. We decided to create a rhythm game where you play as an alien farmer who must collect various resources on Earth. However, be careful: the rules about what needs to be collected change throughout the game.
For this project, there were two of us. I was in charge of the development, music, and sound integration, while my friend handled the graphics. The biggest challenge was creating a level generation system entirely in Fmod, with event callbacks for the appearance of objects as well as rule changes. For the game design, we wanted to create a rhythm game inspired by the decision-making mechanics found in "Papers, Please."
This project was a lot of fun to make, as I had never created a rhythm game before. It's something I definitely plan to do again in the future, as it is very satisfying.
