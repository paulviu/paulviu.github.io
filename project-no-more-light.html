No More Light

31 January 2020 - 2 February 2020

"No more light" is a game created during the Global Game jam 2020. The Global Game jam is an event where you have to create a game on a given theme. In 2020 the theme was "Repair". My team was consisted of 5 people: a 2D artist, a Sound Designer and three Developers, including myself. No more light is an adventure game about repairing earth. Human destroyed earth with their bad actions. Today there is no more light. Can you repair all light sources? The game being in 2D I was able to learn how to use some 2D tools that are in Unity. Moreover, our game was made in URP to have access to the new lights of Unity as well as the VFX graph. I also participated in the design of the story as well as the integration of dynamic music to reinforce the immersion. At the end of the 48 hours, we presented the game in front of the Game Jam jury who gave us the second prize of the Global Gam Jam Toulon!
